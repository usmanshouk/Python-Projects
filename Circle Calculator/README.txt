Requirements:
For this programme to run the local machine must have installed python 3.0 or above.
How to run the code:
Import the code to a compiler and then write the command  python3 followed by location of file in the terminal
and then press enter. It will run the code.
What this code will do:
This algorithm can compute either of radius, diameter, area and circumference of circle, if one of them is given. 
Initially it will ask the user to choose from the menu that what parameter of circle do you have.
Then it will ask the value of that parameter in centimeters.
After that it will compute and display the remaining parameters.
